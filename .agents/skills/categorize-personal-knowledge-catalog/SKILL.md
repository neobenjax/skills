# Categorize Personal Knowledge Catalog Skill

This skill categorizes the data extracted from a document (typically an OKF YAML Frontmatter markdown file generated by the `extract-info-from-pdf` skill) and intelligently merges it into the user's personal knowledge catalog.

## Step 1: Categorize the Information
Determine the path to the OKF YAML Frontmatter markdown file the user wants to process. The user should have provided it in their prompt, as an attachment, or as the output of a previous skill.
If you don't have the file path, ask the user to provide it.

Run the categorization Python script to extract and structure the information into a JSON file:
`python .agents\skills\categorize-personal-knowledge-catalog\scripts\categorize.py <input_markdown_path> extracted_resume.json`

## Step 2: Analyze the Output
Read the generated `extracted_resume.json` file located in your current working directory. The top section of the resume (like personal info) will typically be found under the `"uncategorized"` key.
Next, review the existing markdown files in `knowledge-catalog`. These files use OKF YAML Frontmatter.

## Step 3: Map Information and Preview
Use your LLM capabilities to parse the JSON sections and map the details into the corresponding catalog markdown files (e.g., `experience.md`, `education.md`, `personal_info.md`).
- Present to the user a **summary** of where the extracted information should fit.
- Present a **preview** of every file's intended output (showing exactly how the file will look after changes). 
- **CRITICAL**: Do NOT modify the extracted JSON text, and do NOT modify the actual knowledge catalog markdown files yet.

## Step 4: Ask for Approval
Stop executing tools and explicitly ask the user for approval. Wait for them to agree with the proposed changes.

## Step 5: Apply Changes
After the user approves, apply the changes to the markdown files in `knowledge-catalog`. Make sure to maintain the existing OKF YAML Frontmatter structure and append/merge the data correctly. Once done, inform the user that the operation is complete.
