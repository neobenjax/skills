# Extract Info from PDF Skill

This skill automates the process of extracting text, tables (as markdown), and images from a PDF file using a python script. The extracted information will be placed in a OKF YAML Frontmatter markdown file inside a temporary `extracted-pdf-catalog` folder, named after the attached file.

## Step 1: Ensure Prerequisites
First, verify that the required Python libraries are installed by running:
`python -c "import pypdf, pdfplumber"`

If it fails, install them via the terminal:
`pip install pypdf pdfplumber`

## Step 2: Extract PDF Information
Determine the path to the PDF the user wants to process. The user should have provided it in their prompt or as an attachment.
If you don't have the PDF file or path, ask the user to provide it.

Run the Python script to extract information from the PDF into an OKF YAML Frontmatter markdown file. The script expects the path to the PDF and the directory where the output should be saved.
Execute the following command using relative paths based on your working directory:

`python .agents\skills\extract-info-from-pdf\scripts\extract.py <input_pdf_path> extracted-pdf-catalog`

## Step 3: Output a Summary
After the script finishes execution, read the generated OKF YAML Frontmatter markdown file in the `extracted-pdf-catalog` directory.
Output a summary of the generated file highlighting:
- The title (name of the file)
- The timestamp
- A short summary (generated by you, the AI) of the content extracted from the document.
